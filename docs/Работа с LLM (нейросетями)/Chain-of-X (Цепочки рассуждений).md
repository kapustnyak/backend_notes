## **Расширенный промпт:**

```
Реши следующую задачу, используя Chain-of-Thought:

1. Определи, что именно нужно узнать.
2. Разбей задачу на подэтапы.
3. На каждом шаге делай логический вывод.
4. В конце — итоговый ответ.
```

### **Готовый шаблон Chain-of-X**

```
Используй метод Chain-of-X для решения задачи.

1. Сначала: Определи, что требуется.
2. Затем: Рассуждай пошагово (CoT).
3. При необходимости: Описывай вход (CoD).
4. Задавай себе наводящие вопросы (CGCoT).
5. При желании: Сравни несколько решений и выбери лучшее (SC-CoT).
6. В конце: Чёткий и обоснованный ответ.
```
