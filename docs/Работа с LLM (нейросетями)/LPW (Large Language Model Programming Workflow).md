### **Пример 1 промта с LPW:**

```
Реши следующую задачу программирования, используя метод LPW (Large Language Model Programming Workflow). Следуй этим шагам:
### ФАЗА 1: ПЛАНИРОВАНИЕ
1. Проанализируй задачу.
2. Составь пошаговый план решения.
3. Определи алгоритмы и структуры данных.
4. Проверь план на тестовых примерах (вручную).
5. Укажи ожидаемые промежуточные результаты.

### ФАЗА 2: РЕАЛИЗАЦИЯ
1. Напиши код на Java по плану.
2. Добавь комментарии.
3. Протестируй на указанных примерах.
4. Если есть ошибки — определи источник и предложи исправления.

Используй только Phase 1 сначала. Я дам команду для начала Phase 2.
```

### **Пример 2 промта с LPW:**

```
Задача: Реализовать метод, который определяет, является ли строка палиндромом.

Работай по LPW.

### ФАЗА 1: ПЛАНИРОВАНИЕ

1. Убрать все не-буквенные символы и привести к нижнему регистру.
2. Сравнить строку с её обратной версией.
3. Вернуть true, если они совпадают.

Пример:
"Racecar!" → "racecar" == "racecar" → true

Ожидаемые промежуточные данные:
- Исходная: "Racecar!"
- Обработанная: "racecar"
- Обратная: "racecar"

Жду код, когда я напишу `Phase 2`.
```
